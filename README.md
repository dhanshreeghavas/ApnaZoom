##  Description

**ApnaZoom** is a full-stack real-time video meeting web application inspired by Zoom.  
It enables users to create and join video call rooms, communicate through live audio/video, share screens, and interact in multi-user meeting environments.  
The application uses **WebRTC** for peer-to-peer video communication and **Socket.io** for real-time signaling and room management, providing a smooth and reliable video conferencing experience.

---

##  Features

- User authentication (Sign Up / Login / Logout)
- Create and join meeting rooms
- Real-time video and audio communication
- Multi-user video conferencing
- Screen sharing functionality
- Live chat during meetings
- Secure end-to-end signaling using WebSockets
- Responsive UI for desktop and mobile
